# Proyecto-Progra-juegos

## Protocolo Zero

### High Concept
Protocolo Zero es un juego 3D Top-Down de resolución de acertijos donde el jugador controla a un robot llamado SERA-X, quien debe navegar un mundo post-apocalíptico y lograr escapar de un complejo donde se encuentra encerrado, usando lógica y la resolución de acertijos.

### Game Loop principal
El jugador debe explorar varias áreas cerradas en el complejo donde se encuentra encerrado. Cada nivel funciona como un "piso" distinto del complejo y para avanzar entre ellos se debe explorar el área en búsqueda de llaves, as cuales se consiguen por medio de la resolución de minijuegos y evasión de enemigos.

### Alcance del vertical slice
El vertical slice tendrá un alcande de un único nivel jugable con 3 zonas distintas, dos dedicadas a minijuegos y una a evasión de enemigos, todas con el objetivo de resolver un problema para conseguir una de las llaves.

Así mismo se agregará un menú principal, un menú de pausa, una pantalla de carga y una para la condición de victoria. De momento el juego no cuenta con una condición de derrota, pues en caso de caerse de la plataforma o ser detectado por un enemigo, el jugador solo regresará al punto de inicio en el mismo nivel.
